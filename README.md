AffiliateReports
================

Get emails with aggregated statistics from Adrecord and Tradedoubler.

![Screenshot](https://raw.githubusercontent.com/jongotlin/AffiliateReports/master/screenshot.png)

Install by cloning this repo and install vendors with `composer install`. Configure `app/config/parameters.yml`.
Set up cron to run `php app/console affiliatereports:daily` once every day.
